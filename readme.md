# Graylog cluster docker 

Build your docker conntainers and run a scalable cluster

### Run
`
docker-compose up -d 
`

### Stop
`
docker-compose kill
`

## Delete
`
docker-compose rm -f
`

## Check logs
docker-compose logs -f
